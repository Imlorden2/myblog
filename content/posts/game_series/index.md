## 对 RP 程度的看法

---

RP (Role-Playing) 程度指的是玩家在游戏中对扮演角色的投入程度。

由于**玩家性格**以及**以往游戏经历**的不同，当然也有玩家**自身主观选择**的不同，不同玩家的 RP 程度是不一样的。举一些例子：

```
玩家1:暴躁老哥 - 砍杀、肉鸽游戏历史 - 希望速通 => RP 程度低

思考方式："怎么打赢？怎么最优？"

行为特征：
- 选择数值最高的选项
- 只关心战斗效率
- 对话直接跳过或选最短路径
- 角色就是数据，没有人格

例子：
NPC："请救救我的女儿！"
玩家："给多少钱？不多我不去。"
```



```
玩家2:柔弱妹子 - 经营、养成游戏历史 - 希望感受故事剧情 => RP 程度高

思考方式："我就是这个角色"

行为特征：
- 完全基于角色知识行动
- 宁愿失败也要符合角色
- 为角色写日记、独白

例子：
扮演不识字的野蛮人：
- 看到书：「这些弯弯曲曲的是什么？」
- 即使玩家知道答案，角色也要装不懂
- 智力检定自动放弃
```

从例子就可以看出来，RP 程度只有高低、没有好坏之分，它只是个人的习惯和选择而已。低 RP 会有更高的通关效率和对机制更好的研究；高 RP 会有更好的故事沉浸体验和对游戏细节的体验。但这些都是对游戏本身的体验。

另外，电子游戏的另一个优势在于可以重复体验。有的游戏本身就适合二刷，未尝不可以体验不同的玩法风格思路，从而对游戏有更深的理解和体会。



## 对元游戏问题的看法

---

**元游戏问题**指的是**玩家**利用**角色**不应该知道的信息进行决策。这是游戏难以避免的问题。

尤其是在桌游的场景下，拿桌面 RPG 游戏举例子：

```
DM：“现在你们走进了一片树林，可以感知检定”
玩家1:（思考）DM 让我进行感知检定，说明这片树林肯定有东西
玩家1：“我要仔细搜索这片树林！”
```

这就是玩家1利用了**DM下达命令**这个角色并不知道信息，做出了角色并不会做出的决策。

即使是在电子游戏中，元游戏问题同样存在。最简单的例子其实就是重复游玩的经验，比如新玩家通过查询攻略的方式，得知本关boss巨魔可以用火魔法造成巨额伤害，这也是角色本身不应该知道的信息。这同样也叫做元游戏问题。

为什么元游戏问题是一个问题？它的存在会破坏游戏沉浸感、降低挑战性、新老玩家游戏体验差距被放大...

完全解决元游戏问题是不可能的，但是仍然被采用一些方式降低。比如陷阱位置随机化、敌人配置随机化、减少二周目优势等。而且适度的元游戏可以帮助新手避免一些致命的错误、加快游戏节奏以及增强团队之间的信息传递。
